### ubuntu 安装

#### 更新`apt`索引

```
sudo apt-get update
```

#### 安装最新版本的`Docker Engine`

```
sudo apt-get install docker-ce docker-ce-cli containerd.io

```

### 安装`nginx`

首先一般的安装都是经过如下步骤：

```
docker search XXX
docker pull XXX:版本
```

而`nginx`有官网提供支持

![/img/nginx_pull.jpg](/img/nginx_pull.jpg)

### 查看本地镜像是否已经安装成功

```
docker images
```
### 下载下来镜像运行

```
docker run --name my-nginx -p 9000:80 -d nginx
```
- --name 容器的名称
- -p 9000:80 代表的是将本地端口9000映射到容器
